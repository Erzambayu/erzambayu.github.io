# ThisForYou - Interactive Love Message Website

## Project Overview

Developed an interactive web experience designed to express affection through a beautiful and engaging user interface. The project combines visual elements, animations, and personalized messaging to create a memorable digital gift.

## Key Features

- 💖 Beautiful gradient background with floating hearts animation
- 🎉 Interactive confetti effects when expressing love
- 🎵 Background music that plays during the experience
- 💬 Engaging conversation flow with personalized responses
- 📱 Fully responsive design for both desktop and mobile
- ✨ Smooth animations and transitions
- 🖼️ Cute couple image reveal at the end
- 💕 Customizable messages and sender name

## Technologies Used

- HTML for structure
- CSS for styling and animations
- JavaScript for interactivity
- Responsive design principles
- Animation libraries for visual effects
