# Semangat Sayang - Motivational Website

## Project Overview

Developed a motivational web application designed to help couples track their work goals and motivate each other. The project focuses on creating a supportive environment for achieving professional targets through mutual encouragement.

## Key Features

- 📅 Automatic date display and digital clock
- 💬 Random motivational quotes with 15+ inspirational messages
- ✅ Checklist for tracking goals and targets
- 📊 Progress bar to visualize achievements
- 📝 Ability to add new targets
- 🗑️ Option to remove unnecessary targets
- 🔄 Automatic weekly reset every Sunday
- 🔔 Notification reminders for incomplete targets
- 💾 Local storage to maintain data when browser is refreshed
- 💌 Personal messages

## Technologies Used

- HTML5 for structure
- CSS3 for styling
- Vanilla JavaScript for functionality
- Local Storage API for data persistence
- Web Notifications API for reminders
- Responsive design with mobile-first approach
